Css Loading Animation

發表於 2023-09-07 更新於 2023-09-08
紀錄練習 Css 完成簡易 loading animation 以及在文章內嵌入 codepen 。

Css animation 練習

這個練習可以注意到 margin-left 的設定，讓文字比原本只有 translateY 的上下移動外，還有向左右拓展的動畫效果。


在文章中嵌入 codepen

從網路上找的資源顯示有 hexo-codepen 這個套件可以使用，但是我不管怎麼試，在官方文件中
{% codepen userId|anonymous|anon slugHash theme [defaultTab [height [width]]] %}
↑ 這個使用方式，呈現出來的 codepen 預覽寬高都搞不定，於是再嘗試找找其他資源。

然後在 hexo 文件 中，看到有支援 iframe 的語法

嘗試使用，直接從 codepen embed，選擇 iframe，直接把 src 貼上就如文章開頭一樣成功嵌入了 !
